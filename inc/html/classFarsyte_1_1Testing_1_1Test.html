Detailed Description

The Test object.

Derived from Output String Stream.

The Test object constructor writes appropriate opening text to the XML file associatd with the Suite. The destructor writes appropriate text to the XML file to close the XML element corresponding to the test. Methods on Test objects are available for reporting test conditions that are skipped, test conditions that fail, and errors encountered during testing. It is an error to construct a Test object for a Suite that currently has an active Test object. Be sure that each Test object goes out of scope before the next one is constructed.

Data sent to this object using the << operator will be formatted appropriately and displayed by Bamboo as supporting text in appropriate reporting conditions.
