Detailed Description

The Suite Object.

The Suite object constructor writes appropraite opening text to the XML output stream. The destructor writes appropriate text to the XML stream to close the XML element corresponding to the suite. It is an error to construct a Suite object for a Log that currently has an active Suite object; be sure that each Suite object goes out of scope before the next one is constructed.
